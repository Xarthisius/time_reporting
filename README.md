Install:

python setup.py install

Usage:

  time_reporting.py [date] [hours]                                                                                                                                                                                                                               


   hours - 7 values, for Sunday - Saturday, of hours worked.                                                                                                                                                                                                      
          Default: 0 8 8 8 8 8 0 (40 hour work week M-F)                                                                                                                                                                                                         

   date  - date for overdue time reporting                                                                                                                                                                                                                        
          Example: 01/21/1999
